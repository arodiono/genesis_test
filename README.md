## Run locally

### Install dependencies

```bash
yarn install
```

### Run the app

```bash
# development
yarn start:dev

# production mode
yarn start:prod
```

### Run test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```


---

## Run with docker

### Up container

```bash
yarn dev:up
```
